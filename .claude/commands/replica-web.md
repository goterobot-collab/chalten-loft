# Replica Diseños Web

Analiza la URL que te doy y replica su layout, colores, tipografía y animaciones para el proyecto Chaltén Loft.

## Instrucciones

Cuando el usuario provea una URL, ejecutar:

```
Esta es la landing page $ARGUMENTS. Replica el layout y espaciado, la tipografía, los patrones de animación, y los colores y la paleta de colores. Usa el skill UI UX Pro Max y Tailwind CSS para construir componentes completamente responsive para el proyecto Chaltén Loft. Que los estilos se puedan aplicar al diseño existente del sitio — reemplazando colores, fuentes y layouts donde corresponda — pero manteniendo el contenido y copy actuales intactos.
```

## Uso
```
/replica-web https://url-de-referencia.com
```

## Skills disponibles
- `ui-ux-pro-max` — 161 paletas, 57 combinaciones de fuentes, 67 estilos UI
- `web-reader` — analiza la URL de referencia
- `frontend-design` — reglas de diseño, anti-AI-slop, motion guidelines
